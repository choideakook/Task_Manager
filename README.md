# 업무 분담 매니저

## ✏️ Projec 요구사항

- 프로젝트를 함께 진행하는 맴버간 업무를 할당하고 관리할 수 있다.
    - 한명당 여러개의 업무를 맡을 수 있다.
    - 하나의 업무는 한명의 맴버만 담당할 수 있다.
- 업무의 내용, 처리 기간, 진행 상태를 확인할 수 있다.
- 할당된 업무에 대해서 모든 맴버들은 코맨트를 남길 수 있다.
